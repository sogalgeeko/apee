Ouvrir une image seule :

Menu "Fichier" => Ouvrir une image... (ou Ctrl + o).
Choisir votre image, les tags s'affichent et peuvent être modifiés.
Cliquer sur "Enregistrer" (ou Ctrl + s) pour sauver vos modifications.
Vous pouvez naviguer parmi les autres images contenues dans le même dossier.

Ouvrir un dossier :

Menu "Fichier" => "Ouvrir un dossier..." (ou Ctrl + Maj + o).
Choisir votre dossier, la première image trouvée dans le dossier ou un de ses sous-dossiers d'affiche.
Les tags peuvent être modifiés et sauvegardés.
Vous pouvez naviguer parmi toutes les images contenues dans le dossier choisi ou ses sous-dossiers à l'aide des touches fléchées ou des boutons de navigation " < " et " > ".

Appliquer des tags aux images d'un dossier :

Menu "Chantier" => "Éditer les tags communs".
Vous devez choisir le dossier dont vous voulez éditer les images par lot.
Saisissez les tags souhaitées et validez.
Après affichage de la confirmation, toutes les images contenues dans le dossier choisi (et ses sous-dossiers) sont étiquettées.

Créer un listing au format CSV :

Menu "Chantier" => "Export CSV".
Choisir le dossier dont les images doivent être listées.
Un message vous informe que l'export commence, cliquez sur "Ok" pour le fermer.
Un second message vous informe que l'export est terminé, il apparaît à la racine du dossier choisi sous le nom "listing.csv".

Supprimer l'image active :

Pour supprimer l'image actuellement affichée, cliquez simplement sur "Supprimer".
ATTENTION : l'image est effacée du disque et ne peut être retrouvée.

Quitter l'application :

Menu "Fichier" => "Quitter" (ou Ctrl + q).
